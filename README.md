House Sales in King County, USA 

Overview

This project is part of a Data Analytics final assignment, where I act as a Data Analyst working at a Real Estate Investment Trust (REIT). The Trust is looking to invest in residential real estate, and my task is to determine the market price of a house given a set of features.

Using a dataset of house sales in King County, USA, I analyze and predict housing prices based on attributes such as square footage, number of bedrooms, number of floors, and other key features.




Objectives
Import and clean real-world housing data

Perform data wrangling to prepare the dataset for analysis

Conduct exploratory data analysis (EDA) to uncover patterns and relationships

Develop predictive models to estimate house prices

Evaluate and refine models for better accuracy


Tools & Libraries Used
Python

Pandas, NumPy – data manipulation

Matplotlib, Seaborn – visualization

Scikit-learn – model development and evaluation


Key Steps
Importing Data – Load the King County house sales dataset

Data Wrangling – Handle missing values, duplicates, and data types

Exploratory Data Analysis – Visualize trends, correlations, and distributions

Model Development – Build regression models to predict house prices

Model Evaluation & Refinement – Improve model performance using techniques like cross-validation and hyperparameter tuning


Results
Identified key features influencing house prices (e.g., square footage, location, number of bathrooms)

Built and compared multiple regression models

Delivered a reliable price prediction model for residential real estate investments
